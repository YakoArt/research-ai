# Документация проекта research-ai

Эта папка содержит техническую документацию по проекту.

## Структура

- [`backend/ai-service.md`](./backend/ai-service.md) — описание сервиса
  `backend/src/ai/ai.service.ts`: архитектура, последовательность работы,
  tool-calling, потоковая выдача и диаграммы.

## Соглашения

- Документация пишется на русском языке.
- Диаграммы оформляются в виде блоков [Mermaid](https://mermaid.js.org/),
  которые корректно рендерятся в GitHub, GitLab и большинстве IDE.
- Имена файлов, классов, методов и переменных в тексте оборачиваются в
  обратные кавычки.
- Каждый документ начинается с краткого резюме («что это и зачем»),
  далее идёт описание решения и детали реализации.
